나무 질감 파워포인트 ppt 배경 2종
파워포인트, PPT/강좌  I  2015. 11. 4. 15:33
 

파워포인트 작업에 사용되는 슬라이드 배경은 대부분 단색이나 그라데이션을 사용합니다.

별다른 준비 과정 없이 클릭 몇 번으로 쉽게 사용할 수 있기 때문입니다.

하지만 그만큼 나만의 특별한 무엇을 찾아보기 힘들죠.

평범하고 가벼운 결과물이 되기 쉽다는 말입니다.

 

파워포인트 배경만으로 결과물의 완성도를 높이기는 어렵습니다.

하지만 좋은 분위기를 만들 수는 있죠.

지금 소개하는 나무 질감이 그런 경우입니다.

 

IT기기 리뷰나 개봉기를 보면 유독 자주 사용되는 배경이 마루 같은 원목 질감입니다.

왜냐하면, 나무는 사람과 가장 친근한 소재이기 때문입니다.

친근하다는 것은 거부감 없이 사람들이 좋아하고 잘 어울린다는 말입니다.

뭔가 프레젠테이션의 본질과 잘 통하지 않나요?

 

 

 

두 종류의 나무 질감을 준비했습니다.

첫 번째 나무 질감은 채도 낮은 고급스러운 느낌의 질감입니다.

어두운 명도가 오브젝트를 잘 살려줄 것 같네요.

 

 

 

 

얼마 전에 포스팅한 아이콘 오브젝트를 올려보았습니다.

어두운 명도의 질감이라서 텍스트와 함께 아이콘의 형태가 잘 드러나네요.

 

 

 

 

 

예전에 작업했던 결과물에 적용한 모습입니다.

자칫 차가워 보일 수 있는 기하학적인 도형을 따뜻한 나무 질감이 잘 감싸주고 있습니다.

특히 커피 오브젝트는 나무 질감 배경과 정말 잘 어울리네요.

실제 나무 탁자 위에 커피잔이 나란히 놓여 있는 것 같습니다.

 

 

 

 

 

이번에는 광원이 추가된 약간 붉은 느낌의 나무 질감 배경입니다.

광원이란 전구처럼 빛이 나는 물체를 말합니다.

실제로 보이지는 않지만, 나무 배경 위에 전구가 하나 달려 있을 것 같지 않나요?

처음 소개한 배경이 나무 바닥의 느낌이라면 이번 배경은 광원 효과 때문에 나무 벽처럼 느껴지네요.

 

 

 

 

나무 벽에 뭔가를 주렁주렁 매달듯이 슬라이드에 오브젝트를 배치합니다.

오브젝트를 공처럼 아주 빵빵하게 만들고 그림자를 더 길게 늘어뜨리니 광원 효과가 한층 더 잘 살아나네요.

 

 

 

 

 

커피 콘텐츠가 나무 배경과 너무 잘 어울려서 한 번 더 만들어 봤습니다.

음~ 역시 잘 어울리네요!!
